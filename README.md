# pi-crew

Sub-agent extension for the [pi coding assistant](https://github.com/badlogic/pi-mono).

Status: in development. See `docs/superpowers/specs/2026-04-25-pi-crew-design.md`.

## Install

```bash
pi install git:github.com/leninkhaidem/pi-crew@v0.1.0
```

After install, run `/subagent config` to configure model slots.
